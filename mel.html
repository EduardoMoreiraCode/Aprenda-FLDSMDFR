Visão Geral da Linguagem Mel

A linguagem Mel, desenvolvida para o ambiente FLDSMDFR, foi pensada para o controle de dispositivos em sistemas de manufatura, como braços robóticos. É simples, prática e focada em produtividade.

Sintaxe Básica

Declaração de variáveis:

  • Variável Inteira: x = 10;
  • Variável Float: x = 10.01;
  • Variável Booleana: x = true

Aritmética

  • Soma: 10 + 5 => 15
  • Subtração: 10 - 5 => 5
  • Multiplicação: 10 * 5 => 50
  • Divisão: 10 / 5 => 2
  • Preferência para parênteses: (8+2)/5 => 5
  • Sinais em operações: 10 - (-10) => 20
  • Números decimais: 10 - 5.5 =>

Comparação:

  • Maior: 10 > 5
  • Menor: 5 < 10
  • Maior ou igual: x >= 5
  • Menor ou igual: x <= 10
  • Igual: 5 == 5
  • Diferente: 10 != 5

Lógica Booleana:

  • And (lógica E): 1==1 and 2==2
    1==1 && 2==2
  • Or (lógica OU): 1==1 or 2==2
    1==1 || 2==2
  • Not: !bool

  • Maior: 10 > 5
  • Menor: 5 < 10
  • Maior ou igual: x >= 5
  • Menor ou igual: x <= 10

Estrutura de Dados:

  • If/Else:
                        
    if (1 = 1) {
        print 1; 
    } else { 
        print 0;
    }
  • While:
                        
    while (x < 10) {
        print x;
        x = x + 1;
    }
  • For: 
                        
    for(i = 0; i < 10; i++) {
        print i;
    }
